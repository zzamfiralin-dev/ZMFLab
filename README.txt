
ZMFLab - final site (static)
=================================

Ce include acest pachet:
- site static modern (HTML/CSS/JS) cu dark mode
- logo SVG minimalist integrat (logo.svg)
- 50 produse cu pagini individuale (product_<SKU>.html)
- client-side cart (localStorage) cu sidebar, checkout via WhatsApp sau Email
- imagini folosite de pe Unsplash (link-uri directe)

Pași rapizi pentru GitHub Pages:
1. Creează repository (de ex. username.github.io/zmflab)
2. Upload fișierele din acest pachet în root
3. Activează Pages (branch: main, folder: /)
4. Site live în câteva minute: https://<username>.github.io/zmflab/

Notă: verifică licența imaginilor pentru uz comercial, înlocuiește textele și prețurile dacă e nevoie.
